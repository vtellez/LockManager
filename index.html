Want More?

  • Friendly and intuitive

    Simple and intuitive interface based on the use of wizards, so that you can restore your backup in just a few clicks!

  • Easy integration and improvement

    You can restore any file that is available from a backup. Can extend the functionality to virtual hard disk servers or any other type of resource.

  • Detailed stats

    Get detailed stats for all recoveries made with Recuper@. It also has indicators and generate usage reports.

  • Multi Language Support

    Multi language application, with the possibility of adding support for any language. Currently, you will find the most used languages availables.
